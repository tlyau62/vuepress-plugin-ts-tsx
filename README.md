# vuepress-plugin-ts-tsx

A vuepress plugin that enable ts and tsx support.

## Setup

### Install

```
npm i vuepress-plugin-ts-tsx
```

### Config

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    // ...
    ['vuepress-plugin-ts-tsx'],
  ],
};
```
